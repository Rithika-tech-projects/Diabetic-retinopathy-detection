# Hyper-Personalized Multi-Modal Deep Learning for Diabetic Retinopathy Detection

## 📌 Project Overview
This project focuses on detecting Diabetic Retinopathy using a deep learning-based multi-modal approach that combines retinal images and patient clinical data.

## 🎯 Objectives
- Early detection of diabetic retinopathy
- Improve accuracy using personalized data
- Reduce false negatives in diagnosis

## 🧠 Methodology
- CNN model for image feature extraction
- MLP for clinical data processing
- Fusion layer to combine both modalities
- Softmax classifier for prediction

## 📊 Dataset
- EyePACS Dataset
- APTOS 2019 Blindness Detection Dataset

## ⚙️ Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- NumPy

## 📈 Results
- Improved accuracy compared to image-only models
- Better early-stage detection
- Reduced misclassification

## 🔍 Features
- Multi-modal learning (image + clinical data)
- Personalized prediction
- Scalable model
